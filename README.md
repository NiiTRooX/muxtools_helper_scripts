# muxtools-helper-scripts

Useful functions to use with Muxtools

### If imports don't work

- Add it to an existing repo like this: `git submodule add git@github.com:NiiTRooX/muxtools_helper_scripts.git helper_scripts`
    - It will be cloned into `./helper_scripts`
- Clone it like this: `git clone git@github.com:NiiTRooX/muxtools_helper_scripts.git helper_scripts`
- Or rename the module so it doesn't start with muxtools
